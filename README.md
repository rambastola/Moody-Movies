# MoodyMovies

## Recommend movies based on emotion

Emotion to Genre:<br/>
Sad –> Drama<br/>
Disgust –> Musical<br/>
Anger –> Family<br/>
Anticipation –> Thriller<br/>
Fear –> Sport<br/>
Enjoyment –> Thriller<br/>
Trust –> Western<br/>
Surprise –> Film-Noir
